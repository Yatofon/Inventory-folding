Запуск программы

Открывает терминал Git Bash и создаём адрес в Инвентаре-складывании:

cd Inventory-folding mkdir build cd build

Компилируемый проект

cmake .. cmake --build

Копируем все файлы .dll из /SFML/bin в сборке (всего их 10)

Запускаем

./example.exe

Примечание №1: Проверка ошибок линковки sfml файлов, потому что я запускаю с помощью gcc и Task.json, в остальном, вроде, на самой сборке не ругается

Примечание №2: Я создал нормальную сетку для инвентаря, разве что у инвентаря отсутствует обводка справа и сверху

Примечание №2: Твои файлы я не трогал, впринципе комментарии по методу объясняют, что и для чего нужно, но если надо будет, то всё объясню в тг
