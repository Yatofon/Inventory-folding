Запуск программы

Открываем терминал Git Bash и создаём папку build в Inventory-folding:

cd Inventory-folding
mkdir build
cd build

Компилируем проект

cmake ..
cmake --build

Копируем все .dll файлы из /SFML/bin в build (всего их 10)

Запускаем

./example.exe

Проверь на ошибки линковки sfml файлов, ибо я запускаю с помощью gcc и task.json, в остальном, вроде, на саму сборку не ругается
