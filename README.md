Запуск программы

Открываем терминал Git Bash и создаём папку build в Inventory-folding:

cd Inventory-folding
mkdir build
cd build

Компилируем проект

cmake ..
cmake --build

Копируем все .dll файлы из /SFML/bin в build (всего их 10)

Запускаем

./example.exe


ПРИМЕЧАНИЕ №1
Изначально шрифты лежат отдельно в корневой папке
Для их использования после создания папки build или компиляции проекта необходимо перенести файл "Tokushupikuseru-Regular.otf" в build
Без загрузки шрифтов в build проект скорее всего не запустится

ПРИМЕЧАНИЕ №2
Вся SFML лежит в одноимённой папке SFML.

ПРИМЕЧАНИЕ №3 (хз насколько он нужен, я делал т.к. у меня не было ничего для C++)
Необходимо скачать MinGW:
1) https://www.msys2.org/  (msys2-x86_64-20260611.exe)
                            (устанавливать в папку по умолчанию)
2) Открой MSYS2 UCRT64 и исполни pacman -Syu
2.2) Если появится запрос, закрой окно, открой заново и ещё раз исполни (повторяй пока не увидишь, что все пакеты обновлены)
3) Исполни там же pacman -S --needed mingw-w64-ucrt-x86_64-toolchain
4) Проверка установки: исполни в Git Bash в VSCode g++ --version
